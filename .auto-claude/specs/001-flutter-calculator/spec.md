# Flutter Calculator App 개발

## Overview

Flutter를 사용한 완전한 계산기 앱 개발. 기본 사칙연산과 고급 수학 기능을 제공하는 모던 UI의 계산기.

## Requirements

### 핵심 기능
- 기본 연산: +, -, *, /
- 소수점 지원
- 괄호 연산
- 클리어 (C, CE)
- 백스페이스

### 고급 기능
- 퍼센트 (%)
- 루트 (√)
- 제곱 (x²)
- 음수 전환 (+/-)

### UI 요구사항
- 반응형 레이아웃
- 다크/라이트 테마
- 숫자 패드 그리드
- 연산 히스토리 표시

## Acceptance Criteria

- [ ] 기본 사칙연산이 정확하게 작동
- [ ] 소수점 계산 정확도
- [ ] 에러 처리 (0으로 나누기 등)
- [ ] UI가 여러 화면 크기에서 작동
- [ ] 테마 전환 기능
